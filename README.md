
# 📄 Question\_Paper\_Generator

An automated tool for generating question papers based on predefined rules like difficulty level, marks distribution, and subject topics. Designed to simplify and speed up the process of creating balanced and fair question papers for academic institutions.

## 🔧 Features

* 🗂️ Categorized question bank (by subject, chapter, difficulty, marks)
* 🎯 Customizable question paper format (total marks, number of questions, types)
* 🔁 Randomized and unique paper generation
* 👀 Preview and edit before finalizing
* 📤 Export to PDF or printable format
* 🔐 Secure question bank storage

## 📚 Use Cases

* Schools and colleges for exam paper creation
* Tutors creating mock tests or practice papers
* Competitive exam platforms needing dynamic paper generation

## 🛠️ Tech Stack

* Python 
* Database: Local System
* GUI: Tkinter 


## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/ArhamJamal000/Question_Paper_Generator.git
   ```
2. Set up the environment:

   ```bash
   cd Question_Paper_Generator
   pip install -r requirements.txt
   ```
3. Run the project:

   ```bash
   python main.py
   ```
